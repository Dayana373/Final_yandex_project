# Проект: Клиент под контролем (Выпускной проект)

Проект представляет собой простую модель машинного обучения для прогноза оттока клиента (ушел/не ушел). Модель использует градиентный бустинг CatBoostClassifier. Точность на тестовом наборе данных состовляет __~93%__.


## Как открыть тетрадку:

Открыть с графиками plotly:
  1. [Перейдите по ссылке на nbviewer](https://nbviewer.org/github/Dayana373/Portfolio_Yandex/blob/4518223c1509ffa941e87acc8017c1c75f7f97b1/final_project.ipynb)

## Данные:

Используются [датасет](https://disk.yandex.ru/d/8hb_9fAZatuyMw) Яндекс.Практикума. Содержит до 7043 пользователей с информацией об их тарифах и услугах.
